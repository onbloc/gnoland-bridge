# Gno.land Bridge

A web frontend for bridging GNOT and wrapped GNOT between Gno.land and EVM chains through the Union protocol.

## Features

- Gno.land and EVM wallet connection with Adena and MetaMask
- GNOT bridge flows backed by Union ZKGM routes
- Direct Gno to EVM token initialization flow
- Live bridge activity dashboard
- Light and dark theme

## Development

Requirements: Node.js 20+ and pnpm.

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```
